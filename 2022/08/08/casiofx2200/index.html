用 CASIO fx-220 计算器求算数平均数、方差的方法

注意！

下文提到的计算器为 CASIO fx-220 计算器，其他计算器的操作可能有所不同。

用计算器求平均数

提示：如果你找不到键位，可以看这里。

例：求下列数据的平均数：$55,54,51,55,53,53,54,52$。

  1. 按开机键 AC，打开计算器。
  2. 按 MODE· 键选择进入SD模式。
    • 如果显示屏中有FIX或SCI指示符，请先按 MODE9 键。
  3. 按 SHIFTAC 键开始输入数据。
    • 输入 $55$，按 DT 确认。
    • 输入 $54$，按 DT 确认。
    • 输入 $51$，按 DT 确认。
    • ……（以此类推）。
  5. 按 SHIFT∑x÷SHIFTn= 即可计算出这一组数的平均数。
  6. 上述数据的平均数为 $53.375$。

注意：按两次 DT 可以输入两次相同的数据。

用计算器求方差

提示：如果你找不到键位，可以看这里。

例：求下列数据的方差：$55,54,51,55,53,53,54,52$。

  1. 按开机键 AC，打开计算器。
  2. 按 MODE· 键选择进入SD模式。
    • 如果显示屏中有FIX或SCI指示符，请先按 MODE9 键。
  3. 按 SHIFTAC 键开始输入数据。
    • 输入 $55$，按 DT 确认。
    • 输入 $54$，按 DT 确认。
    • 输入 $51$，按 DT 确认。
    • ……（以此类推）。
  5. 按 SHIFTσnSHIFTx²= 即可计算出这一组数的方差。
  6. 上述数据的方差为 $1.734375$。

注意：按两次 DT 可以输入两次相同的数据。

键位提示

  1. DT 键对应在 M+ 上。
  2. ∑x 键对应在 5 上。
  3. n 键对应在 6 上。
  4. σn 键对应在 8 上。
  5. x² 键对应在 √ 上。
